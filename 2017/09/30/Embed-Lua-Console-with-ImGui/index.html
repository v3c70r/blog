Embed Lua Console with ImGui

2017-09-30

A new Lua console is added to the fluid engine.


Lua is a common scripting language in game engines thanks to its small size and low memory footprint. Though I’m quite new to this language, but I have seen the potential it has to enrich my engine in the future.
Anyway, the interactive Lua consoles used famous Dear ImGui and simplified the Console example. I redirected the inputs and outputs to Lua state and it’s turned out to work pretty well.

The implementations are in these two files
GL4Framework/src/lib/console/LuaConsole.h
GL4Framework/src/lib/console/LuaConsole.cpp

I’ll post more about the application of Lua scripting in the engine in the future.
